他跪坐其间，吻了红尘

有一蓑烟雨 何不任平生

他们身后，跟着那艘湿漉漉的旧船，它搁浅多年 残破不堪，如今满载骸骨，尽然又能乘风破浪了——《全球高考》英雄不该是这样的结局，不该被辜负，不该这样离开 ——《穿堂惊掠琵琶声》他们走得不紧不慢，袍摆轻扫却了无尘埃，山道弯袅，岁月漫长，停停走走便是遥遥一生了

想对你好，希望你能活在温暖的阳光里，正直，谦逊，纯真，勇敢，去爱或被爱，不必乞求得不到的东西，不必提前经受生活的磨难，而我来守护你不可救药的浪漫和理想主义.阳光依然干净，星河依然灿烂，世界也依然在长久深情中缓缓朝前走，于是时间奔流，得以见证人间，在漫长岁月里所有永恒和不朽——《黑天》.我尊重世界上一切存在，但不代表我要永远沉浸其中——

我连魂魄都是黑的，唯独心尖上一点点，血还是红的。用它护着你，我愿意.殿下，风光无限的是你，跌落尘埃的也是你，重点是你，而不是怎样的你。奇怪，说出对不起之后，每个人理所当然的，对别人口中的没关系深信不疑—

长阶血未尽
人的动摇始于第一次心软—
